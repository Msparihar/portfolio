## Dark Fantasy Color Palette (Elden Ring + Witcher + Dark Souls)

```css
/* BACKGROUNDS */
--dt-bg-base:        #0d0b07;   /* Near-black, warm tint */
--dt-bg-surface:     #1a1510;   /* Dark brown surface */
--dt-bg-elevated:    #2c2318;   /* Raised panels */

/* GOLD / AMBER (Elden Ring golden runes) */
--dt-gold-bright:    #f9c043;   /* Max brightness gold */

--dt-gold-dim:       #8b6914;   /* Dark gold */
```

## CSS Patterns

### Stone/Parchment Grain Texture (CSS-only, no external files)
```css
.dt-surface {
  background-image: 
    url("data:image/svg+xml,%3Csvg viewBox='0 0 200 200' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='n'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.65' numOctaves='3' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23n)'/%3E%3C/svg%3E"),
    linear-gradient(135deg, #1a1510, #2c2318);
  background-blend-mode: overlay;
}
```

### Pulsing Gold Rune Glow
```css
@keyframes runeGlow {
  0%, 100% { text-shadow: 0 0 8px #f9c043, 0 0 20px #c9a84c; }
  50%       { text-shadow: 0 0 16px #f9c043, 0 0 40px #ee8a05, 0 0 60px #8b6914; }
}
.dt-rune { color: #f5d87a; animation: runeGlow 3s ease-in-out infinite; }
```

### Chamfered Gothic Panel (Souls/Dragon Age style)
```css
.dt-panel {
  clip-path: polygon(
    12px 0%, calc(100% - 12px) 0%, 100% 12px,
    100% calc(100% - 12px), calc(100% - 12px) 100%,
    12px 100%, 0% calc(100% - 12px), 0% 12px
  );
  background: linear-gradient(160deg, #1a1510 0%, #0d0b07 100%);
  border: 1px solid #4b3d2a;
  box-shadow: 0 0 20px rgba(0,0,0,0.9), inset 0 0 12px rgba(0,0,0,0.6);
}
```

## Key Insight
Chamfered `clip-path` panels + Cinzel font + gold glow = instant Souls-like aesthetic. The grain texture trick (inline SVG feTurbulence) costs zero HTTP requests.
