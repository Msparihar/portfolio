# v2.0.0 — 3D Interactive World (Genshin-style Exploration)

**Status:** VISION / LONG-TERM  
**Created:** 2026-04-09

---

## Overview

Transform the static wallpaper into a **3D interactive world** that users can explore. Instead of a flat background image, the wallpaper becomes a navigable 3D scene — zoom in on the soldier by the bonfire, climb the Ghibli mountain, fly over King's Landing.

## Concept

The desktop OS sits ON TOP of a 3D world. Users can:
- **Zoom in/out** on the wallpaper — reveals hidden details, Easter eggs
- **Pan/rotate** the camera to explore the scene
- **Click on objects** in the world — the soldier, the bonfire, a door, a signpost → triggers actions (opens apps, reveals lore, starts mini-games)
- **Hide desktop icons** to go into "exploration mode" — full 3D world
- **Toggle back** to desktop mode with icons overlaid

## Per World

### Elden Ring
- 3D dark corridor scene (based on wallpaper)
- Zoom into the bonfire → find soapstone messages
- Click the soldier → About page (character sheet)
- Click doors → open project pages
- Walk down the corridor → discover more rooms (projects)

### Ghibli
- 3D pastoral landscape with hills, cottage, trees
- Climb the mountain → see portfolio overview from above
- Enter the cottage → find the journal/codex
- Follow a cat through the meadow → discover hidden projects
- Watch sunrise/sunset as time passes

### GoT
- 3D Winterfell courtyard / King's Landing panorama / The Wall
- Fly between regions (dragon flight for Dragonstone)
- Enter the Great Hall → project gallery
- Visit the Maester's tower → codex/lore
- Walk The Wall → timeline of experience

## Technical Stack
- **Three.js** or **React Three Fiber (R3F)** — 3D rendering in React
- **GLTF models** — lightweight 3D scenes per world
- **Raycasting** — click detection on 3D objects
- **Orbit controls** — camera pan/zoom/rotate
- **Post-processing** — bloom, depth of field, fog
- **Performance:** LOD (level of detail), frustum culling, texture compression
- **Progressive:** Start with 2.5D parallax (CSS), graduate to full 3D

## Phases

### Phase 1: 2.5D Parallax Wallpaper
- Wallpaper split into depth layers (foreground, midground, background)
- Mouse movement creates parallax effect
- Scroll/pinch to zoom in/out
- Clickable hotspots on the wallpaper

### Phase 2: Simple 3D Scene
- Three.js scene replaces wallpaper
- Basic camera controls (orbit, zoom)
- Clickable objects linked to apps
- Low-poly aesthetic to keep it fast

### Phase 3: Full Exploration
- Detailed 3D environments per world
- Character movement (WASD or click-to-move)
- NPC encounters in 3D space
- Seamless transition between exploration and desktop mode

## Notes
- This is a long-term vision — v2.0.0 at earliest
- Start with Phase 1 (parallax) which is pure CSS + minimal JS
- 3D models can be generated with AI tools or sourced from Sketchfab
- Performance is critical — must work on average hardware
- Mobile: simplified 2D version with touch controls
