## Overview

The portfolio OS simulator ships with multiple "worlds" — complete visual experiences that transform the entire UI. Each world has its own palette, typography, window chrome, wallpaper, ornamental elements, icons, and effects. Same core content (projects, blog, contact), completely different aesthetics.

## Easter Eggs

### Dark Souls Phantoms (Elden Ring / Dark Souls world)
- **Concept:** Translucent ghost figures that wander across the desktop, inspired by Dark Souls' phantom system
- Ghosts appear as semi-transparent white/blue silhouettes
- Behaviors:
  - Drift across the desktop slowly
  - Stop and "interact" with windows (ghost clicking, typing)
  - Occasionally two ghosts engage in combat (sword clashes, dodge rolls)
  - Fade in/out with Souls-style translucency effect
- **Messages:** Glowing orange messages on the desktop ground (Souls soapstone style)
  - "Try jumping" / "Amazing portfolio ahead" / "Praise the sun!" / "Don't give up, skeleton"
  - Users can rate messages (upvote/downvote like Souls)
- **Trigger:** Appears after user has been on Elden Ring world for 2+ minutes
- **Technical:** CSS animations + requestAnimationFrame, pre-recorded movement paths, random spawn intervals
- Could also adapt to GoT world (ghosts of fallen characters)
- **Bloodstain replay:** Click glowing red spots on desktop to see a phantom's "death replay" (funny animations)

### Forest Creatures (Ghibli world)
- **Concept:** Animated creatures wander the desktop, inspired by Studio Ghibli's living worlds
- Creatures:
  - **Cats** — Jiji-style black cats and calicos walking across the desktop, pausing to sit and groom
  - **Birds** — Swallows and sparrows flying across the screen occasionally
  - **Soot sprites** — Little black dust bunnies (from Totoro/Spirited Away) appear near corners or on idle
  - **Butterflies / fireflies** — Drift over the wallpaper, fireflies at "night" if time-based
  - **Totoro silhouette** — Appears briefly at screen edge during rain/weather effects (rare Easter egg)
- **Trigger:** Creatures start appearing after 30 seconds on Ghibli world
- **Technical:** CSS sprite animations, randomized spawn points and paths, lightweight SVG or CSS-only creatures
- **Interaction:** Hovering near a creature makes it pause and look at cursor, clicking makes it scurry away

### House Animals (GoT world)
- **Concept:** Each GoT region has its own wandering house creature
- **The North:** Direwolves padding across the desktop, ravens perching on window corners
- **King's Landing:** Lions prowling, hawks circling above
- **Night's Watch:** Crows circling and landing, ghostly direwolf in the distance
- **Dragonstone:** Small dragons flying across the screen, ember trails behind them
- **Trigger:** Appears with the seasonal cycle, creature changes when region changes
- **Technical:** Same animation system as Ghibli creatures, region-specific sprite sets

### Living Worlds Pattern
Every world has its own "life" system — this is a **consistent cross-world pattern**:
- **Elden Ring:** Phantoms wandering + soapstone messages
- **Ghibli:** Forest creatures (cats, birds, soot sprites)
- **GoT:** House animals (direwolves, lions, crows, dragons)
- **Default/Cyberpunk:** Could have drones, glitch artifacts, or digital particles
- This pattern makes each world feel alive and rewards visitors who stay

### Seasonal Cycle Discovery (GoT world)
- Not announced anywhere — visitors discover it by staying on the site
- First transition includes a subtle toast: "Winter is coming..." or "The seasons are changing..."
- After seeing all 4 regions, unlock a "Seasons Explorer" panel in Display Options

### Future Easter Egg Ideas
- **Screensaver:** Animated character running across desktop after idle (like PostHog's dinosaur)
- **Konami Code:** Triggers a special hidden world or animation
- **Click counter:** After N clicks on desktop, reveal a hidden message
- **Terminal secrets:** Hidden terminal commands that unlock content

## Design Principles (Cross-World)

### Frosted Glass Windows
- All app windows should be semi-transparent (70-80% opacity) with backdrop blur
- Wallpaper imagery bleeds through windows, creating depth and immersion
- Taskbar and sidebar also use translucency
- This applies to ALL worlds — the wallpaper is a key visual element, not hidden behind opaque UI
- Per world: Elden Ring uses darker tint (#0d0b07CC), Ghibli uses warm cream (#faf6edCC), GoT varies by region

### AI-Generated Wallpapers
- Every world desktop MUST have a proper AI-generated wallpaper, not just gradient fills
- Wallpapers set the mood more than any other single element
- Current wallpapers: Elden Ring (dark corridor with bonfire), Ghibli (pastoral meadow with cottage), GoT North (The Wall with aurora), GoT Night's Watch (fortress with torches)

## Architecture Notes

- worlds.js: delta pattern (BASE_THEME_VARS + per-world overrides merged in applyWorld())
- Worlds are mutually exclusive with themes (localStorage stores one or the other)
- GoT seasonal cycle needs a timer system in the world store (setInterval + crossfade logic)
- Phantom system: separate module, only loaded when Dark Fantasy world is active
- Assets: AI-generated imagery via Pencil for mockups, production assets TBD (may need custom illustrations)
