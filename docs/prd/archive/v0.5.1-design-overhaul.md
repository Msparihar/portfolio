## Version Goal

Complete the visual design system for all three worlds (Default, Dark Fantasy, Ghibli) in Pencil, then implement the designs accurately in code. The current Dark Fantasy CSS implementation is a placeholder — it needs a full rebuild to match the Pencil mockups.

---

## Background

v0.5.0 shipped the Worlds architecture (worlds.js, WorldPicker, applyWorld, CSS class system) but the Dark Fantasy visual implementation was CSS-only color swaps — user rejected it as inadequate. We then created proper design mockups in Pencil:

- **Dark Fantasy:** 9 pages (P1-P8 + P2b) — style guide, desktop, ornamental windows, components, file manager, terminal, mail/about/gallery, taskbar/sidebar, transitions/states
- **Ghibli:** 3 pages — style guide, desktop overview, components
- **Default (current):** No formal design system exists — it's the v0.3.1 neutral dark theme

The Pencil designs include ornamental elements (⚜ ✝ † ✦), pillar borders, AI-generated fantasy imagery, gold dividers, chamfered panels, gothic typography — things that can't be achieved with just CSS variable swaps.
