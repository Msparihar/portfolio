## Long-Term Vision (2-year roadmap)

This is not a portfolio — it's a virtual world / interactive experience. Something people explore for hours, share with friends, and keep coming back to discover new things. Think VR-lite in the browser.

**Roadmap:**
- **v0.4.0** — Desktop OS polish (PostHog-level) — immediate next step
- **v0.5.0** — Worlds system + first 2-3 themed worlds
- **v0.6.0** — 3D elements, animations, deep easter eggs
- **v0.7.0** — Sub-worlds (projects as explorable places)
- **v0.8.0+** — VR/immersive elements, spatial audio, multiplayer?
- Each version is deployable and impressive on its own
- Ship incrementally, don't try to do it all at once

## Vision

Evolve the portfolio from "terminal with windows" into a full-fledged desktop OS experience like PostHog's homepage. Not copying PostHog, but reaching that level of polish and interactivity.

## Bigger Vision: "Worlds" — Themed Experiences

The portfolio/agency site becomes an explorable multiverse. Each "world" is a completely different visual + interactive experience wrapping the same core content (projects, blog, contact, about).

**Potential Worlds:**
- **Cyberpunk/Terminal** — current dark hacker aesthetic
- **Adventure/Isometric** — PostHog-style pixel art, garden/farm, warm tones
- **Game of Thrones** — medieval map, castles, parchment, heraldry icons
- **Elden Ring** — dark fantasy, golden runes, fog, stone UI
- **Harry Potter** — magical parchment, floating candles, Marauder's Map navigation
- **Space** — sci-fi cockpit, starfield, holographic UI
- **Studio Ghibli** — soft watercolors, nature, whimsical

**Why this works:**
- Shows creative range (great for agency positioning)
- Each world is a portfolio piece itself
- Visitors remember the experience, not just the resume
- "Explore my worlds" is a far stronger hook than "view my projects"
- Could double as agency site (manishsingh.tech) — clients pick a world that fits their brand

**Core Experience Principles:**
- **Deep exploration** — users should be able to explore for hours and keep discovering new things, like PostHog
- **Animations everywhere** — transitions, hover effects, micro-interactions, parallax, entrance animations
- **3D elements** — Three.js / React Three Fiber for immersive world elements (rotating planets, floating castles, magical particles, etc.)
- **Easter eggs at every layer** — hidden interactions, secret commands, clickable objects that reveal surprises
- **Discoverable depth** — the more you explore, the more you find. Reward curiosity.
- **Not a static site** — it should feel alive, interactive, like a game world you can walk through

**Sub-worlds / Project Links:**
- Personal projects (ReddiChat, LinkFlow, novel reader, etc.) become sub-worlds within the main world
- Users can "jump" from the main world into a project's sub-world — same theme carries over
- Each project isn't just a card — it's a place you can visit and explore
- The portfolio becomes a hub/portal connecting all personal projects under one unified experience

**Architecture thought:** Each world = a theme config (colors, wallpapers, icons, fonts, animations, sounds) + a layout variant + 3D scene. The desktop OS shell stays the same, but everything inside transforms. Sub-worlds (projects) inherit the parent world's theme.
