## Overview

Each world in the portfolio OS has its own **in-universe lore system** — a collection of books, journals, maps, and codex entries that tell an original story about the world. Visitors don't just see a themed portfolio — they discover a living world with its own history, written by the user (who is also a novel author).

The lore system serves dual purpose:
1. **Portfolio narrative** — explains the projects, skills, and journey through in-universe storytelling
2. **Creative showcase** — demonstrates writing ability directly within the portfolio experience

---

## Core Components

### 1. Codex / Bestiary
- **What:** Encyclopedia of the world's creatures, characters, artifacts, and locations
- **Per world:**
  - **Elden Ring:** Item descriptions (cryptic, souls-style), boss entries, weapon lore, location memories
  - **Ghibli:** Nature guide (creatures, plants, spirits), character profiles, location postcards
  - **GoT:** Maester's compendium, house histories, creature bestiary, weapon/armor entries
- **UX:** Opens as an app window (like a book), searchable, categorized by type
- **Content format:** Short entries (50-200 words each), souls-style — lore revealed through objects, not exposition

### 2. Travel Log / Journal
- **What:** First-person journal entries about visiting each world, written in-character
- **Per world:**
  - **Elden Ring:** Tarnished's journal — entries about discovering each project as a "location" in the Lands Between
  - **Ghibli:** Nature diary — warm, reflective entries about building each project as tending a garden
  - **GoT:** Raven scrolls / Maester's letters — reports about events (projects) in each region
- **UX:** Chronological, scrollable, dates tied to actual project creation dates
- **Tone:** Each world has a distinct narrative voice

### 3. Moon Log / Seasonal Chronicle
- **What:** Tracks the passage of time and seasonal changes within the world
- **Per world:**
  - **Elden Ring:** Moon phases tied to the Erdtree cycle, celestial events
  - **Ghibli:** Season wheel — spring blossoms, summer fireflies, autumn leaves, winter snow
  - **GoT:** The Long Night approaches, seasonal transition chronicle, winter/summer counter
- **UX:** Visual calendar/wheel that shows current phase, history of transitions
- **Easter egg:** The moon log actually tracks how long the visitor has been on the site

### 4. World Map
- **What:** Interactive map of the world, with labeled regions/locations
- **Per world:**
  - **Elden Ring:** Lands Between map — each project is a "location" on the map (like Grace sites)
  - **Ghibli:** Hand-drawn watercolor map — projects as villages/landmarks in a pastoral landscape
  - **GoT:** Westeros map — projects placed in different regions, seasonal overlay shows current season
- **UX:** Clickable locations open the corresponding project, hover shows brief description
- **Technical:** SVG-based or canvas-rendered, styled per world

### 5. Dictionary / Glossary
- **What:** In-universe definitions of terms, technologies, and concepts
- **Per world:**
  - **Elden Ring:** "Ashes of War" = deployed projects, "Runes" = skills/experience, "Grace" = milestones
  - **Ghibli:** "Seeds" = project ideas, "Bloom" = launched projects, "Roots" = core skills
  - **GoT:** "Houses" = companies worked at, "Banners" = technologies, "Alliances" = collaborations
- **UX:** Alphabetical, cross-referenced with codex entries
- **Purpose:** Maps real portfolio content to in-universe metaphors

### 6. Folklore / Story Chapters
- **What:** Original serialized story about the world — WHY it exists, its origin, its purpose
- **Per world:**
  - **Elden Ring:** The Shattering — how the portfolio came to be, told as a souls-style creation myth
  - **Ghibli:** The Spirit's Garden — a gentle fable about growing things and nurturing ideas
  - **GoT:** A Song of Code and Fire — epic chronicle of building, battling bugs, and conquering deployments
- **UX:** Chapter-by-chapter, beautifully typeset, with decorative elements per world
- **Content:** Original creative writing by the user (novel author showcase)
- **Chapter structure:** Prologue + 5-8 chapters + Epilogue per world

## App Integration

The lore system manifests as **apps within the desktop OS:**

### Book App (primary lore reader)
- Opens as a window styled like a physical book (leather-bound for Elden Ring, picture book for Ghibli, illuminated manuscript for GoT)
- Table of contents on the left, content on the right
- Page-turn animations
- Bookmark system (localStorage)
- Reading progress indicator

### Map App
- Full-screen or windowed interactive map
- Zoom/pan controls
- Location pins with tooltips
- Connected to project data from portfolio.json

### Codex App
- Grid/list view of all codex entries
- Filter by category (creatures, locations, items, characters)
- Search functionality
- "Discovered" vs "Undiscovered" entries (gamification — entries unlock as user explores the site)

## Gamification Layer

### Discovery System
- Codex entries start as "???" (undiscovered)
- Entries unlock as visitors:
  - Visit different pages/apps
  - Stay on the site for certain durations
  - Find Easter eggs
  - Switch between worlds
  - Click on hidden elements
- Progress bar: "X/Y entries discovered"
- Completionist reward: unlock a secret world or special message

### Achievements
- "First Steps" — opened the terminal
- "Explorer" — visited all apps
- "Loremaster" — read all codex entries in one world
- "World Walker" — switched between all worlds
- "Phantom" — encountered a phantom (Elden Ring world)
- "Season Watcher" — witnessed all 4 GoT seasonal transitions
- "The Author" — read all folklore chapters

## Writing Style Guide (per world)

### Elden Ring
- Cryptic, fragmented, never fully explains
- "This [item] was once wielded by..." style
- Short paragraphs, poetic cadence
- Reference: Dark Souls item descriptions, Elden Ring lore

### Ghibli
- Warm, gentle, reflective
- Nature metaphors for everything
- "In the quiet of the forest, where the old code grows..."
- Reference: Miyazaki film narration, children's storybooks

### GoT
- Epic, political, dramatic
- Multiple perspectives (different "authors" of entries)
- "The maesters record that in the year of..."
- Reference: GRRM's writing style, in-universe histories like "Fire & Blood"
